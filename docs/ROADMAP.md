# NeonVersum Investor App – Roadmap

## Projektziel

Die NeonVersum Investor App ist ein geschütztes Investor-Produktdashboard.

Die App ist kein Shop.

Sie zeigt später:

- Investitionen
- Dokumente
- Unternehmensupdates
- Partnerstatus
- Holderstatus
- persönliche Vorteilscodes
- Benutzerprofil

Der Einkauf findet auf der separaten NeonVersum-Shop-Homepage statt.

---

## Benutzerarten

### Kundentyp

- Privatkunde
- Firmenkunde

### Rollen

- Investor
- Händler
- Partner
- Administrator

Die Rolle wird später durch NeonVersum geprüft und freigegeben.

---

## Holderbonus

Partner oder Händler können einen Holderrang erhalten.

Beispiele:

- Bronze
- Silber
- Gold
- Platin

Die Investor App zeigt:

- aktuellen Rang
- Holderstatus
- persönlichen Vorteilscode
- Gültigkeit des Codes

Der Rabatt wird ausschließlich im separaten Shop eingelöst.

## Technische Regeln

- Keine echten Passwörter in GitHub speichern
- Keine Investorendaten direkt in HTML oder JavaScript speichern
- Keine geheimen Schlüssel öffentlich hinterlegen
- Keine Rabattcodes fest im Frontend eintragen
- Jede Seite separat entwickeln und testen
- Nach jedem Commit auf Smartphone und Desktop prüfen
